A practical guide to applied machine learning

Jeremy Gore

Tessella Autumn Conference 2015-10-05

To say machine learning is a big subject is a big understatement, but it may be easier than you think.

Using python and scikit-learn, we'll walk through a machine learning process from start to finish discussing when to apply machine learning, the most successful algorithms, best practices, and even a little philosophy.

Random Forests

Fun Fact:

Decision Jungles are a relative of Random Forests that use Directed Acyclic Graphs (DAGs) instead of binary trees. This allows for much smaller models which generalize slightly better. They were developed by Microsoft.

The Kinect controller for XBox uses Random Forests to analyze what it's seeing. You do the math.
